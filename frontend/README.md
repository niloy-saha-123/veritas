# Veritas Frontend

Frontend dashboard for Veritas - AI-powered documentation verification.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
